Tingkat Aktivitas Gunung Api di Indonesia

by. Shafa Dara Illona

Berdasarkan BNPB (2018), berikut adalah penjelasan mengenai status level gunung api

  1. LEVEL I (Normal) :Tidak ada aktivitas gunung api yang berarti. Kondisi aman.
  2. LEVEL II (Waspada) :Terjadi perubahan gejala yang diakibatkan oleh aktivitas magma, tektonik, dan hidrotermal. Larangan beraktivitas di sekitar gunung.
  3. LEVEL III (Siaga) :Gunung api sedang berpotensi menimbulkan bencana. Mengosongkan daerah sekitar.
  4. LEVEL IV (Awas) :Gunung api berpotensi akan Meletus. Mengosongkan wilayah yang berada pada Kawasan Rawan Bencana sesuai petunjuk PVMBG

    Ring Of Fire

    By. Shafa Dara Illona

    Ring of Fire adalah istilah yang digunakan untuk menggambarkan busur gunung api aktif yang berukuran sangat panjang, yakni sekitar 40.0000 km (24.900 mil). Busur gunung api ini terbentuk sebagai manifestasi yang dihasilkan dari akitivitas beberapa batas lempeng tektonik, meliputi Lempeng Pasifik, Juan de Fuca, Cocos, Indian-Australian, Nazca, North American, dan Philippine (National Geographic, 1996). Diperkirakan, sebanyak 70% gunung api yang ada di dunia – sekitar 450 gunung api – berlokasi pada busur ini. Indonesia sendiri menyumbang lebih dari 200 gunung api yang tersebar dari wilayah Sumatera, Jawa, Bali, dan pulau-pulau di timur Indonesia..

    Mitigasi Gunung Api

    By. Dinda Rinjani & Shafa Dara

    Mitigasi dalam letusan gunung api adalahsebagai berikut:

    Pra letusan

    1. Mengenali daerah sekitar gunung api untuk menentukan daerah yang aman untuk mengungsi ketika terjadi letusan
    2. Membuat persiapan perencanaan penanganan bencana
    3. Menyiapkan kebutuhan dasar
    4. Memantau status gunung api secara berkala

    Pada saat terjadi letusan

    1. Tidak berada pada wilayah yang termasuk ke dalam Kawasan Rawan Bencana, aliran lembah, maupun sungai
    2. enghindari tempat terbuka agar terhindar dari abu vulkanik.
    3. Menggunakan masker untuk menutup mulut dan hidung. Tidak menggunakan lensa kontak.
    4. Mengenakan pakaian yang menutupi tubuh.

    Pasca letusan

    1. Menjauhi wilayah yang terdampak hujan abu vulkanik.
    2. Tidak berkendara pada wilayah yang terdampak abu vulkanik.
    3. Membersihkan timbunan debu vulkanik pada atap rumah.

    Sumber Informasi: ESDM.go.id
